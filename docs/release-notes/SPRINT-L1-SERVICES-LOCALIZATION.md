# Sprint L1 Services Localization

## Scope

Localized the owner Services module in English, Greek, German, Spanish and Turkish.

## Included

- Page heading and summary metrics
- Search, category and status filters
- Loading and empty states
- Service list labels and statuses
- Add/edit service dialog
- Validation, confirmation and toast messages
- Accessible labels for edit and remove actions

## Business logic

No Supabase queries, payloads, filters, routes or service-management behavior were changed.
